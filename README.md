< Main functions of this App >
1. Add, update, or delete schedule.
2. Show weekly schedules in Table.
3. Select Repeat schedule or not every week.
4. Show Notification(Alarm) before 1 hour of schedule.


< Used Android API >
1. GridView
2. SQLiteDatabase
3. Service
4. Notification
